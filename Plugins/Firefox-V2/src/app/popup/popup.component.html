What is Scrippy?

Right click hacks: Ok, maybe not...

Scrippy is a Chrome and Firefox browser add-on that stores SQL statements to aid in the testing of websites for code injection vulnerabilities. Think cheat sheets but all you do is right click for them.

This tool is made for web developers, to quickly test web sites for basic code injection vulnerabilities. You must have a legal right (Authorisation) to carry out any testing, doing so without this authorisation can be a criminal offence under local and international law. by using this tool you acknowledge the above statement and take sole responsibility for your actions.

Scrippy dose not replace the need for a more in depth security review by professionals.
